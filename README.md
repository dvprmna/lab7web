# Lab7Web - Praktikum 1: PHP Framework (CodeIgniter 4)

## Deskripsi
Repository ini berisi hasil praktikum **Modul Praktikum Pemrograman Web 2**  
Membuat website sederhana menggunakan **CodeIgniter 4** dengan konsep **MVC**.

## Tujuan Praktikum
- Memahami konsep dasar Framework
- Memahami arsitektur MVC (Model - View - Controller)
- Membuat program sederhana dengan CodeIgniter 4

## Teknologi yang Digunakan
- Laragon
- PHP 8.3
- CodeIgniter 4
- HTML5 + CSS3

## Cara Menjalankan
1. Pastikan Laragon sudah **Start All**
2. Buka browser dan ketik:  
   `http://localhost/lab11_ci/ci4/public/`

## Halaman yang Sudah Dibuat
- Home
- Artikel
- About
- Contact
- FAQ

Semua halaman menggunakan **layout yang sama** sesuai modul.
